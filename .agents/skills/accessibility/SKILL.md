# Accessibility Compliance

## Use this skill when
- Designing the UI and injecting dynamic AI content.

## Instructions
1. **Semantic Markers:** Use `<main>`, `<section>`, and `<button>` instead of generic `<div>` click handlers.
2. **Live Regions:** Assign `aria-live="polite"` to the AI summary box so screen readers announce new insights.
3. **Color Contrast:** Maintain a 4.5:1 ratio using Tailwind’s high-contrast pastel palettes.
